Workflow 2: School Request for Information

The Resilience Project — CRM Workflow Documentation for Dynamics 365 Migration • April 2026

Summary: A school requests more information about our programs. The system captures the contact and organisation, then branches based on school size. Large schools (500+ students) enter the sales pipeline directly. Smaller schools enter an automated nurture sequence with timed follow-up emails.

1. Trigger

A school submits a "Request for More Information" form on the TRP website.

3. Business Process

3.1 Identify or create the contact and organisation

  • Same rules as the general enquiry workflow: link to existing org or create new, one active contact per email.
  • Assign ownership using state-based routing (NSW/QLD → Brendan, others → Laura; preserve existing dedicated SPMs).
  • Track which forms the contact and organisation have engaged with.

3.2 Branch based on school size

Path A: Large School (500+ students)

Outcome: Create a sales opportunity

These are high-value prospects. A deal is created so a salesperson can manage the relationship directly.

  • Deal name: "2027 School Partnership Program"
  • Stage: "New"
  • Pipeline: "New Schools"
  • Expected close: 1 week from submission
  • Only created if the school is new (no existing dedicated SPM)

Follow-up is manual — the assigned salesperson works the deal.

Path B: Smaller School (under 500 students)

Outcome: Enter automated nurture sequence

Smaller schools receive a series of timed follow-up emails to maintain engagement without requiring manual sales effort.

  • Contact lifecycle stage set to "Lead", status "Hot"
  • Must not create duplicate nurture entries if the same contact submits again

Follow-up is automated — see comms sequence below.

4. Automated Nurture Sequence (Path B)

Smaller schools receive the following timed communications:

TimingCommunicationCondition
ImmediatelyConfirmation — "Thanks for your interest in our School Wellbeing Program"Always sent
1 week laterFollow-up — checking in, any questions?Only if the school hasn't progressed in the pipeline
2 weeks laterFinal follow-up — "Have more questions about our program?"Only if still not progressed. Deal interest is downgraded to "Cold".

Email content is tailored by school size — schools under 350 students receive different copy to those with 351+.

Business requirement: Follow-up emails must automatically stop if the sales team has already advanced the school's opportunity (e.g. the school has progressed to a live info session or a call). Only schools that haven't engaged further should receive the automated nudges.

5. Owner Assignment

Same rules as Workflow 1 (General Enquiry) for schools:

The same owner is also used as the "reply-to" address on automated emails, so responses go to the right person.

7. Key Business Rules

  1. 500-student threshold: Large schools are high-value and go directly into the sales pipeline. Smaller schools get automated nurture to maintain engagement efficiently.
  2. No duplicate contacts: One active contact per email address.
  3. No duplicate nurture entries: If a contact has already entered the nurture sequence, don't create a second one.
  4. Conditional follow-ups: Automated emails must stop if the school has progressed in the pipeline (e.g. registered for an info session or had a call).
  5. Size-tailored messaging: Smaller schools (under 350) receive different email copy than mid-size schools (351+).
  6. No deals for existing partnerships: If a school already has a dedicated SPM, don't create a new deal.

8. Current Implementation Notes

Note for Dynamics 365 migration: Our current CRM (Vtiger) does not have a native lead nurture or marketing automation capability. To work around this, we use an Event Registration as a proxy — smaller schools are "registered" to a standing event, and CRM workflow rules trigger timed emails based on the registration date. This is a platform workaround, not a business requirement. In Dynamics 365, this should be replaceable with a proper marketing automation journey / lead nurture sequence.
