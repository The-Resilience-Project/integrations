# Business Journeys

High-level maps showing how individual flows connect across the full lifecycle of an engagement. Each journey links to the detailed flow docs for every stage.

## Journeys

| Journey | Stages | Description |
|---------|--------|-------------|
| [Schools](./schools-journey.md) | 7 | Full school partnership lifecycle from enquiry to assessment |
| [Conference](./conference-journey.md) | 4 | Conference lead capture through to prize pack fulfilment |
| [Enquiries](./enquiries-overview.md) | 4 | How school, workplace, early years, and general enquiries relate |

## How to Read These Maps

Each journey map shows:

- **Stages** as boxes in a Mermaid flowchart — each stage corresponds to one [business flow](../index.md)
- **Arrows** showing the typical progression (though not every engagement follows every stage)
- **Form IDs** where a Gravity Form triggers the stage
- **API version** badges (v1, v2, or both) indicating which endpoint handles the stage

Click through to any stage's flow doc for the full VTAP endpoint sequence and CRM field changes.
